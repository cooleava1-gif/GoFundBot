# V1.0 版本
具备基本的选基功能，可根据限定条件挑选基金（从数据库获取数据）；输入基金代码或名称，可以查看基金的详细信息（直接调用API）。

- **前端 (Vue 3):** 负责基金详细数据的展示。
- **后端 (Python Flask):** 负责处理天天基金 API 接口，管理数据库。
- **数据库 (SQLite):** 将基金信息以文件形式存在本地。

需要改进的地方：
- 界面无法显示基金名称，只显示“未知基金”
- 不能选择净值走势的展示时间段
- 基金基础信息用中文
- 单位净值和累计净值混杂
